# yaml-practice
This repository contains YAML practice files and examples for learning and experimenting with YAML syntax and structures.

## Purpose

- Learn YAML basics: key-value pairs, lists, nested structures
- Practice YAML for configuration files (e.g., Kubernetes, Ansible, CI/CD)
- Keep a personal reference of YAML patterns and examples


## Usage

- Open any YAML file to read or edit
- Test YAML syntax using online validators or CLI tools
